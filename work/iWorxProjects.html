    Plugin:
    Created a plugin for an open source program called OpenSesame to work in conjunction with software provided by iWorx Systems, Inc. OpenSesame is used to design social science experiments, and the iWorx software collects and displays physiology data. The plugin establishes a client connection from OpenSesame to a server ran by the iWorx software. The plugin allows experiments in OpenSesame to send messages, data, and time stamps of events to the iWorx software and display them on the screen. This allows the physiology data collected during experiments to be synchronized to what was happening in the social science experiment. Previously, separate charts were recorded and then later compared. This plugin allows all of the data to be centralized in a single program. The plugin was written in Python, and the server portion was written in C++. The Python portion was written entirely by myself and another employee contributed to most of the C++ portion.

    This plug-in was released in the company's newsletter and mentioned in an article.

    iWire-PO2-100 Pulse Oximeter:
