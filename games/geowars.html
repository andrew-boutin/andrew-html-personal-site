Geo Wars Dev Info

  • Technology:
    This game was developed using the Unity Engine . All coding was done in C# using MonoDevelop for an IDE. All team communication was done over Skype and the Kongregate API was utilized to allow high scores to be saved and linked to Kongregate user accounts.

    Dev. Team Structure:
    There was only one other team member on this project who was in charge of game concepts/artwork. He came up with the game, how the mechanics worked, etc. I was the only coder and built the entire game in the engine based on the requirements.

    My Role:
    This was my first game developed using Unity and first team project via only on-line communication. I was the only coder so I built the project from start to finish. Some of the things required to do this included:

    • Physics simulation for player, enemies, and bullets
    • Collision detection
    • Weapon system
    • Artificial Intelligence (AI) for enemy decisions
